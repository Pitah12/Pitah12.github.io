    Utilidad de diagnóstico de redes.

    -n no realiza resolución DNS inversa. Agiliza el analisis.
    -T Velocidad del escanero. Del 0 al 5 siendo 0 lento y 5 rápido pero más ruidoso.
    -sS Escaner TCP SYN.
    -sU Escaner UDP.
    -O Detección del sistema operativo.
    -sC --script=default.
    -nO Para guardar el output en un archivo.
    --script vuln, discovery, auth, etc.
    -p Puertos. -p22 -p1-65535 -p- (todos los puertos)

    --open Solo enseñar puertos abiertos.
    --min-rate No mandar paquetes más lentos que el valor especificado (por segundo).

    Ejemplo: nmap -p- -n -T5 -sS 10.10.11.111 -nO puertos

    Ejemplo: nmap -p- --open -sS --min-rate 5000 -vvv -n -Pn 10.10.11.149 -oN scan

    Ejemplo: nmap -p22,80 -sCV 10.10.11.149 -oN targeted

    Escanear por fuerza bruta las posibles rutas de una URL.

    -c Output con colores.
    -t Número de conexiones simultaneas.
    -hc Ocultar resultados con código de respuesta espeficicado (Hide Code).
    -hw Ocultar resultados con el número de palabras especificado (Hide Words).
    -hh Ocultar resultados con el número de letras espeficiado. —hh=854 oculta los resultados con 854 caracteres.

    Ejemplo: wfuzz -c -t 200 --hc=404 -w /usr/share/dirbuster/wordlists/directory-list-2.3-medium.txt 10.10.11.143/FUZZ

    Herramienta de análisis de red. Navaja suiza para los hackers.

    -n Sólo IPs, sin resolución DNS
    -l Modo escucha
    -v Verbose
    -p Puerto

    Ejemplo: nc -nlvp 443

    Subir o descargar archivos a través del protocolo SSH.
    -r para subir carpetas (recursive).
    -i añadir archivo de identificación (id_rsa)

    (Subir archivo) scp archivo_local usuario@servidor:/ruta
    (Descargar archivo) scp usuario@servidor:/ruta/archivo archivo_local

    Ejemplo: scp -r CVE-2021-4034 www-data@10.10.11.104:/var/www/html/

    Herramienta de recuperación/crackeo de contraseñas.

    Ejemplo: john —wordlist=/usr/share/wordlists/rockyou.txt archivo_hash
